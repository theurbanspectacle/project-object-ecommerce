# project-object-ecommerce

## Getting Started
- Create .env file with 
- DB_NAME=
- DB_USER=
- DB_PASSWORD=

Then 'npm install'

## Scripts
    "start": Start the production server
    "watch": Start the development server
    "seed": Seed the database
    "init:database": Init the database
